# 知能の本質とは何か？
  
このリポジトリは、「知能の本質とは何か」という大きな問いに対して、コードを動かしながら考えていく一連のシリーズです。  
近年のAIは驚くべき性能を発揮していますが、「そもそも知能とは何なのか」という根本的な問いは依然として開かれたままです。  
生物の脳に目を向けると、世界中の巨大プロジェクトがスーパーコンピュータと精緻なモデルを駆使して脳の再現に挑んでいますが、その規模ゆえに、知能の発現に本当に欠かせない仕組みと、生物進化の偶然による細部とを切り分けることが必ずしも容易ではありません。  
  
このシリーズでは、巨大さや精密さではなく、「シンプルさ」と共にこの問いに迫ります。  
手元の環境でも動く規模のミニマルなモデルから出発し、何を足し、何を引くと知能らしい振る舞いが立ち上がってくるのかを、一歩ずつ確かめていきます。  
各記事に対応するコードはすべて Google Colaboratory で動かせる形で公開されており、読むだけでなく、ご自身の手で動かし、パラメータを変え、洞察を深めていただくことが可能です。

シリーズの第一回では、大脳皮質を手がかりに、セル・オートマトンと古典的な人工ニューロンを組み合わせたミニマルなネットワークを構築します。  
投射ニューロンと介在ニューロン、興奮性ニューロンと抑制性ニューロン、恒常性、馴化、ヘブ則という五つの要素を順に導入していくことで、絶え間ないダイナミクスと記憶らしき挙動がどのように立ち上がってくるのかを観察していきます。

シリーズが進むにつれて扱う要素は増えていきますが、出発点はいつもシンプルな構造です。気軽に動かし、考え、議論にご参加いただければ幸いです。
